CS 140

  • Designed and implemented a single process, multi-threaded operating system in C with priority scheduling that runs on an 80x86 processor architecture

COOL Compiler

CS 143

  • Developed a compiler in C++ for the COOL programming language, which compiles into MIPS assembly language
  • Segmented the project into 4 parts: lexical analyzer, parser, semantic analyzer, and code generator
  • Leveraged Flex and Bison unix utilities to write a parser that achieves LALR parsing of context-free grammars
